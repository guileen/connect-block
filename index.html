Best browser compatible solution

connect-block is built upon nodejs and connect, use to show below screen to specified browsers.

Basic usage

    var block = require('connect-block');
    connect(
        //block IE6, IE7, IE8.
        block('IE8'),
        function(req, res){
          res.end('You have passed.');
        }
    ).listen(3000);

Use with express

    app.configure(function(){
        ...
        // block IE6, IE7, IE8
        app.use(block('IE8'));
        app.use(express.bodyParser());
        app.use(express.methodOverride());
        ...
    });
the parameter of block could be 'IE', 'IE6', 'IE7', 'IE8', 'IE9', 'IE10' or 'RegExp'. And the output was localized, currently support Chinese and English. Fork on github to make it support more languages and browsers.
